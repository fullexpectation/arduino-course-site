# 第一阶段：光之魔法师 (数字 I/O 基础) ✅ *已完成*

- **[[1.1 点亮第一盏灯\|1.1 点亮第一盏灯]]**
    - 知识点：Arduino板卡认知、`setup`与`loop`、`pinMode`、`digitalWrite`。
    - 项目：点亮板载LED（Hello World）。

- **[[1.2 闪烁与延时\|1.2 闪烁与延时]]**
    - 知识点：`delay`、程序执行顺序。
    - 项目：SOS 求救信号（三短三长三短）。

- **[[1.3 按钮的逻辑\|1.3 按钮的逻辑]]**
    - 知识点：`digitalRead`、`if-else` 判断语句。
    - 项目：按下按钮灯亮，松开灭（门铃）。

- **[[1.4 逻辑运算\|1.4 逻辑运算]]**
    - 知识点：布尔运算（与 `&&`、或 `||`、非 `!`）、变量基础。
    - 项目：核弹发射井（必须两个按钮同时按才亮灯）。

# 第四阶段：智能机械师 (动作、传感与结构化) ✅ *已完成*

- **[[4.1 舵机动起来\|4.1 舵机动起来]]**
    - 知识点：舵机原理、库的概念 `#include <Servo.h>`、对象实例化。
    - 项目：雨刷器、简单的机械臂。

- **[[4.2 映射的魔法\|4.2 映射的魔法]]**
    - 知识点：`map()` 函数（将 0-1023 映射为 0-180）。

- **[[4.3 状态机与模式\|4.3 状态机与模式]]**
    - 知识点：`switch-case` 语法。
    - 项目：电子宠物（不同模式有不同动作和声音）。

- **[[4.4 超声波测距\|4.4 超声波测距]]**
    - 知识点：HC-SR04 原理、物理公式、`pulseIn`。
    - 项目：超声波测距仪（串口显示）。

- **[[4.5 魔法收纳术 (函数)\|4.5 魔法收纳术 (函数)]]**
    - 知识点：自定义函数 `void` vs `int`、封装代码。
    - 项目：智能感应垃圾桶（用函数重构代码）。
