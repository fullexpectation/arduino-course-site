# **🚀 Arduino 创客大师之路：完整教学大纲**


# **第一阶段：光之魔法师（数字 I/O 基础）** ✅ *已完成*

*   **1.1 点亮第一盏灯**

    *   知识点：Arduino板卡认知、`setup`与`loop`、`pinMode`、`digitalWrite`。

    *   项目：点亮板载LED（Hello World）。

*   **1.2 闪烁与延时**

    *   知识点：`delay`、程序执行顺序。

    *   项目：SOS 求救信号（三短三长三短）。

*   **1.3 按钮的逻辑**

    *   知识点：`digitalRead`、`if-else` 判断语句。

    *   项目：按下按钮灯亮，松开灭（门铃）。

*   **1.4 逻辑运算**

    *   知识点：布尔运算（与 `&&`、或 `||`、非 `!`）、变量基础。

    *   项目：核弹发射井（必须两个按钮同时按才亮灯）。

  

# **第二阶段：能量掌控者（模拟 I/O 与 变量）** ✅ *已完成*

*   **2.1 呼吸灯**

    *   知识点：`analogWrite` (PWM)、占空比概念。

    *   项目：模拟睡眠呼吸效果。

*   **2.2 旋钮控制**

    *   知识点：电位器、`analogRead` (0-1023)、模拟输入。

    *   项目：旋钮调光灯。

*   **2.3 感知光线**

    *   知识点：光敏电阻、分压电路原理（浅讲）。

    *   项目：智能路灯（天黑自动亮）。

  

# **第三阶段：节奏大师（循环、数组与声音）** ✅ *已完成*

*   **3.1 循环的力量**

    *   知识点：`for` 循环（初始值、条件、增量）。

    *   项目：流水灯（跑马灯）。

*   **3.2 随机的乐趣**

    *   知识点：`random()`、随机种子 `randomSeed`。

    *   项目：电子骰子、随机闪烁。

*   **3.3 蜂鸣器与声音**

    *   知识点：有源 vs 无源蜂鸣器、`tone`、`noTone`。

    *   项目：制作防空警报（结合 For 循环频率渐变）。

*   **3.4 音乐播放器**

    *   知识点：**数组**（Array）的应用、查表法、宏定义 `#define`。

    *   项目：播放《超级玛丽》或《孤勇者》。

  

# **第四阶段：智能机械师（动作、传感与结构化）** ✅ *已完成*

*   **4.1 舵机动起来**

    *   知识点：舵机原理、库的概念 `#include <Servo.h>`、对象实例化。

    *   项目：雨刷器、简单的机械臂。

*   **4.2 映射的魔法**

    *   知识点：`map()` 函数（将 0-1023 映射为 0-180）。

    *   项目：旋钮控制机械臂。

*   **4.3 状态机与模式**

    *   知识点：`switch-case` 语法。

    *   项目：电子宠物（不同模式有不同动作和声音）。

*   **4.4 超声波测距**

    *   知识点：HC-SR04 原理、物理公式、`pulseIn`。

    *   项目：超声波测距仪（串口显示）。

*   **4.5 魔法收纳术（函数）**

    *   知识点：自定义函数 `void` vs `int`、封装代码。

    *   项目：智能感应垃圾桶（用函数重构代码）。

# **第五阶段：视觉指挥官（显示与交互升级）** 🆕 *下一站*

*   **5.1 屏幕初体验（I2C LCD1602）**

    *   **理由**：终于可以脱离电脑看数据了！

    *   知识点：I2C 通信简介（只讲接线：SDA/SCL）、液晶屏库的使用 (`LiquidCrystal_I2C`)、坐标系 (列, 行)。

    *   项目：**“你好世界”滚动字幕**。

*   **5.2 可视化测距仪**

    *   知识点：综合应用（超声波 + LCD）。

    *   项目：**便携式尺子**（屏幕实时显示距离）。

*   **5.3 红外遥控（IR Remote）**

    *   **理由**：最受孩子欢迎的控制方式，像电视遥控器一样。

    *   知识点：红外接收、16进制代码（Hex）、`IRremote` 库。

    *   项目：**万能遥控器**（按1开灯，按2舵机转，按3播音乐）。

  

# **第六阶段：环境守护者（高级传感器）**

*   **6.1 温湿度监测（DHT11）**

    *   知识点：环境数据采集、库的依赖安装。

    *   项目：**家庭气象站**（LCD 显示当前温度和湿度）。

*   **6.2 综合逻辑挑战**

    *   知识点：多条件判断、阈值控制。

    *   项目：**智能大棚系统**（温度高了转动舵机（开窗），土干了亮红灯（报警））。

  

# **第七阶段：时间领主（进阶逻辑 - 视情况选讲）**

  

1.  **第一阶段：Delay 的局限**（5分钟）

    *   演示“因为睡觉所以按按钮没反应”。

2.  **第二阶段：外部中断 (External Interrupt)**（25分钟）

    *   **急诊室模式**。

    *   Pin 2 / Pin 3。

    *   实战：炸弹紧急拆除按钮。

3.  **第三阶段：Millis (软件多任务)**（30分钟）

    *   **看手表模式**。

    *   实战：呼吸灯 + 旋钮打印（并行处理）。

4.  **第四阶段：定时器中断 (MsTimer2)**（20分钟）

    *   **生物钟模式**。

    *   实战：主程序 Delay 5秒，LED 依然稳定闪烁。

5.  **总结与挑战**（10分钟）

    *   任务：做一个“有些神经质的机器人”。

        *   **心跳**（Timer）：LED 每秒闪一次，雷打不动。

        *   **痛觉**（External）：按按钮，蜂鸣器立马叫。

        *   **思考**（Millis）：串口不断打印 "Thinking..."。

        *   **发呆**（Loop）：主程序里偶尔加个 `delay(2000)` 模拟死机。

  

# **第八阶段：终极创造者（电机驱动与小车）**

*   **8.1 让轮子转起来（直流电机）**

    *   知识点：直流电机 vs 舵机、电机驱动模块（L298N 或 Arduino扩展板）。

    *   项目：电风扇（控制电机转速）。

*   **8.2 组装战车**

    *   知识点：硬件集成、电池供电管理。

    *   项目：组装底盘。

*   **8.3 避障小车（最终 BOSS）**

    *   知识点：综合算法（测距 -> 判断 -> 转向）。

    *   项目：**全自动避障车**（整合超声波、电机驱动、函数封装）。

*   **8.4 遥控小车（可选）**

    *   知识点：结合红外遥控。

    *   项目：**红外遥控赛车**。
